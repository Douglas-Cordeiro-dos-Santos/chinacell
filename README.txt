Projeto feito para criar um sistema de gestão para pequenos comercios. No início foi pensado para ter diversas funções, como por exemplo, gerenciamento de estoque, controle financeiro, cadastro de clientes, emissão e recepção de notas ficas. Porém priorizamos a parte do estoque para ter algo mais palpável a primeiro momento e depois ir implementando essas outras features aos poucos.

Nesse programa estamos utilizando Python e MySql, e para criação das interfaces, Qt6.
